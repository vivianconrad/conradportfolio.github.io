Vi Conrad's Portfolio Design System

About

This website documents the design system I developed for my personal portfolio. While I generally prefer neutral colors, especially blacks, this project pushed me to explore more colorful and creative design elements.

Foundations

This section covers the design foundations for the design system, including typographical hierarchy, color palette, and logo variations.

  • Typography: Guidelines for the typographical hierarchy used throughout the portfolio
  • Color Palette: A vibrant and creative color palette
  • Logo Variations: Different logo versions and their appropriate usage.

Components

This section outlines the reusable components needed to build the portfolio website:

  • Navigation Bar: Design and functionality for the website's main navigation.
  • Footer: Layout and design of the website footer.
  • Cards: Reusable card designs for displaying content.

Page Patterns

This section provides visual guidelines for the layout of key pages within the portfolio:

  • Home Page: Creative and engaging design for the homepage, showcasing key projects and introducing myself.
  • About Me Page: Detailed layout for the about me page, reflecting my journey, skills, and personality.
